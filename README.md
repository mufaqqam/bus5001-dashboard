# BUS5001 — Council Incident Management Dashboard

A lightweight static web dashboard for visualising community 
service incidents, built as part of BUS5001 Cloud Platforms 
and Analytics (Assessment 1).

## What's in this repo

- `index.html` — Static dashboard that reads from Azure Blob Storage
- `summary.json` — Curated incident summary generated by the Azure Logic App pipeline

## Architecture

CSV Upload → Azure Blob Storage (raw-incidents)
          → Azure Logic App (validates + processes)
          → Azure Blob Storage (public-dashboard/summary.json)
          → This dashboard (reads summary.json and renders it)

## Live Dashboard
(https://bus5001councilsa.z44.web.core.windows.net)

## Azure Services Used

- Azure Blob Storage (data storage + static website hosting)
- Azure Logic Apps (automated workflow)
